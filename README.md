# ResearchLens — Gap Analysis Suite

An AI-powered research paper analyzer. Upload PDFs and get:
- Paper summary & key findings
- Research gap detection (by type & severity)
- Future work extraction
- Generated research questions
- Gap intensity heatmap
- Related papers via Semantic Scholar
- One-click research proposal generator

**Powered by Google Gemini 1.5 Flash — completely free.**

## Live site
`https://<your-username>.github.io/ResearchLens/`

## Setup
1. Go to [aistudio.google.com](https://aistudio.google.com/app/apikey) → sign in with Google → Create API key (free)
2. Open the site → paste your key in the banner → Save for session

## Deploy to GitHub Pages
1. Create a new repo named `ResearchLens`
2. Upload `index.html`, `.nojekyll`, `README.md`
3. Go to Settings → Pages → Branch: main / (root) → Save
4. Live in ~60 seconds at `https://<your-username>.github.io/ResearchLens/`
